Creating a Space

Select the "Create a Space" button on the splash page. You will be redirected to a new Space where your will see your unique temporary Space ID. Input this Space ID in the peer device that you would like to share with.

Joining a Space

On the splash page input the Space ID recieved from creating a Space, then select the "Join" button. You will be redirected to the associated Space where you can immediately begin sharing files and text.

Exiting a Space

Within the Space page select the "Exit" button located on the top right side of the screen. You will be returned to the splash screen.

Sending file(s)

Within the Space page select the "Paperclip" button located on the bottom left side of the screen. Select one or more files to include in the message. Finally, select the "Arrow" Button on the bottom right side of the screen.

Sending text

Within the Space page select the text input located on the bottom center of the screen. Paste or type all of the text you would like to send. Finally, select the "Arrow" Button on the bottom right side of the screen.

Accessing files

Files can be downloaded by left-clicking on the associated file link within the message.

Accessing text

Text can be copied by left-clicking the text within the message. It will be automatically copied to your clipboard.
