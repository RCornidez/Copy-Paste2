Project Summary

The Copy/Paste application was created to make it easier to share text and files between devices. Typically one would email themselves something between personal devices. But in situations where you might be working with a remote system it's now possible to send text content or a file instead of typing it manually. This system works with any device with a browser and internet access.

Functionality and Features

  • Real-time Communication Establishment: Utilizes WebRTC for creating real-time communication channels between devices. Users can start a session and receive a unique session ID.
  • Session Joining Capability: Allows a single user to join an existing session using the session ID, Allowing two users to connect and share data.
  • Text and File Transfer: Enables users to send text messages and files within a session through dedicated buttons and modal forms.
  • Firebase integration as a Signaling Server: Manages WebRTC signaling data using Firebase, with Firestore Database temporarily storing session details.
  • Secure WebRTC Data Channels: Ensures the privacy and security of data transferred between connected peers.
  • User Interface Elements: Incorporates session management, content display, and interactive modal forms for user actions.
  • Dynamic Content Handling: JavaScript functions dynamically manage session IDs, data transfer, and user interactions.
  • Accessibility and Responsiveness: Designed for usability across various devices and screen sizes.

Challenges and Solutions

I ran into a few roadblocks while working with the WebRTC API and Firebase Library, mainly because I was new to these tools. Also, using Vanilla JavaScript added some extra challenges, especially since it doesn't have built-in global state management. But after some solid debugging, I figured it out. I used sessionStorage to keep track of variables, set up polling to check for updates, and organized the client-side JavaScript into a single, large block of code. This setup made it easier to manage global values throughout the app.
